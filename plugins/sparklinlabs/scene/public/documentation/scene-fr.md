# Editeur de scène

***

### Contrôles

Il y a deux modes de caméra: 3D et 2D. Vous pouvez change le mode dans la barre en haut de l'éditeur.
Vous pouvez également changer la caméra par défaut dans les options.

 * Mode 3D

L'éditeur de scène utilise les mêmes contrôles que Minecraft.

**Z/S** pour avancer/reculer.

**Q/D** pour aller à gauche/droite.

**Espace/Shift** pour monter/descendre.

Utilisez le **bouton de milieu** ou **Alt+le bouton gauche** pour orienter la vue.

 * Mode 2D

Utilisez le **bouton de milieu** ou **Alt+le bouton gauche** pour déplacer la vue.

Utilisez la **molette** ou **+/-** pour zoomer.

***

### Raccourcis claviers

Utilisez **Ctrl+N** pour créer un acteur et **Ctrl+P** pour créer un préfab.

Utilisez **F2** pour renommer un acteur, **Ctrl+D** pour le dupliquer et **Suppr** pour le supprimer.

Utilisez **F** pour center la vue sur l'acteur sélectionné.
